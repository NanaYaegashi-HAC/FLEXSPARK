## 担当箇所
**・プレイヤー**

移動、死亡処理などプレイヤー全般の処理

**・ギミック**
LightningFlash：自分が跳ね返したパックが触れると相手の陣地がフラッシュで一瞬見えなくなる処理
CoreSizeBiggerC:自分が跳ね返したパックが触れると相手のコアが大きくなる処理

**・UI**
タイトルロゴやHPバー、リザルト画面などの各シーンのUI作成、配置、システム(HPバー、スコア)。
